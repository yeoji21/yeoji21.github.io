값의 범위가 왜 그렇게 되는지 이유

돈계산할때는 long, double쓰면 안되고 BigDecimal 써야 함

```java
float number = 0f;
for(int i=0; i<10; i++){
    number += 0.1f
}
System.out.println(number);
```
실행하면 1.0f가 나와야 하지만 결과로 1.0000001이 찍힘  

BigDecimal로 변경한 코드  
```java
BigDecimal number = BigDecimal.ZERO;
for(int i=0; i<10; i++){
    number = number.add(BigDecimal.valueOf(0.1));
}
System.out.println(number);
```
